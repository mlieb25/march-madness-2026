# 🏀 March Madness ML 2026

A full-pipeline machine learning system for predicting the NCAA Tournament — built as a public portfolio project and deployed as an interactive Streamlit app.

---

## Features

| Page | Description |
|---|---|
| **Home** | Hero overview, top-line metrics, navigation cards |
| **Bracket** | Full 68-team interactive bracket with probability overlays |
| **Team Explorer** | Per-team efficiency profiles, seed context, and championship odds |
| **Simulator** | 10,000-run Monte Carlo bracket simulation with distribution charts |
| **Upset Detector** | Matchups where the model diverges most from seed-implied expectations |
| **Explainer** | SHAP-powered model interpretability for any head-to-head matchup |
| **Experiment Lab** | Model development history, CV results, and live tournament scoring |

---

## ML Pipeline Summary

**Data sources**
- Barttorvik historical efficiency stats (2008–2026)
- FiveThirtyEight historical tournament results (2008–2025)
- NCAA NET rankings (2026 bracket seeding)
- Kaggle March ML Mania 2026 dataset

**Feature engineering** — 12 pairwise features per matchup:

| Type | Features |
|---|---|
| Differentials | adjOE, adjDE, Barthag, SoS, WAB, Tempo |
| Ratios | adjOE, adjDE, Barthag, SoS, WAB, Tempo |

**Models**

| Phase | Model | Log Loss | Brier |
|---|---|---|---|
| Phase 2 | Logistic Regression (baseline) | 0.5039 | 0.1657 |
| Phase 3 | XGBoost (108-combo GridSearchCV) | 0.6013 | 0.2049 |
| Final | Ensemble (60% XGB + 40% LR) | — | — |

Train/test split: 2011–2023 train / 2024–2025 holdout.

**Simulation** — Vectorized NumPy Monte Carlo over all C(68,2) = 2,278 pairwise probabilities. 10,000 simulations complete in < 3 seconds.

## Quick Start

### 1. Install dependencies

```bash
cd march-madness/app
pip install -r requirements.txt
```

### 2. Run the app

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`.

### 3. (Optional) Run with a custom port

```bash
streamlit run app.py --server.port 8502
```

## Data Requirements

The app reads CSVs from `../data/` relative to `app/`. Ensure the following files exist before running:

| File | Source | Notes |
|---|---|---|
| `ncaa_net.csv` | NCAA.com / manual scrape | 68-team NET rankings for 2026 |
| `barttorvik_historical.csv` | barttorvik.com | All seasons 2008–2026 |
| `ml_training_data.csv` | Generated by pipeline | Pairwise matchup features |
| `ml_inference_data_2026.csv` | Generated by pipeline | All 2,278 2026 matchup pairs |
| `xgb_predictions_2026.csv` | Generated by pipeline | XGBoost win probabilities |
| `baseline_predictions_2026.csv` | Generated by pipeline | Logistic regression probs |

Kaggle files (in `data/march-machine-learning-mania-2026/`) are used for live tournament scoring once the 2026 tournament begins; the app runs safely without them.

## Configuration

`app/.streamlit/config.toml` sets a dark theme matching the app's navy palette. Key overrides:

| Setting | Value |
|---|---|
| `primaryColor` | `#1976D2` (bright blue) |
| `backgroundColor` | `#0d1b2a` (deep navy) |
| `secondaryBackgroundColor` | `#1b2838` (mid navy) |
| `textColor` | `#e8ecf0` (off-white) |
